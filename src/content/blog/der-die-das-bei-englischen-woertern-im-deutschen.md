Die deutsche Sprache hat drei grammatische Geschlechter:

- **der** (maskulin)
- **die** (feminin)
- **das** (neutral)

Wenn englische Wörter ins Deutsche übernommen werden (sogenannte **Lehnwörter**), brauchen sie ebenfalls einen Artikel. Leider gibt es **keine einzige feste Regel**, die immer funktioniert. Es gibt jedoch **klare Muster und hilfreiche Hinweise**, die das Lernen deutlich einfacher machen.

In diesem Leitfaden erklären wir die **wichtigsten Regeln, hilfreiche Tipps und häufige Ausnahmen**, damit Lernende das grammatische Geschlecht englischer Lehnwörter im Deutschen besser einschätzen können.

---

# 1. Das Geschlecht folgt oft dem deutschen Äquivalent

Eine der zuverlässigsten Strategien ist folgende:

**Ein Lehnwort übernimmt häufig das Geschlecht des entsprechenden deutschen Wortes.**

### Beispiele

| Englisches Lehnwort | Deutsches Äquivalent | Geschlecht |
| ------------------- | -------------------- | ---------- |
| der Computer        | der Rechner          | maskulin   |
| die App             | die Applikation      | feminin    |
| die E-Mail          | die Post             | feminin    |
| das Baby            | das Kind             | neutral    |

### Merktipp

Wenn du die **deutsche Übersetzung** kennst, kannst du oft **denselben Artikel verwenden**.

Beispiel:

- **die App** → weil **die Applikation** feminin ist.

---

# 2. Wortendungen geben oft Hinweise auf das Geschlecht

Viele englische Lehnwörter folgen **typischen deutschen Mustern bei den Wortendungen**.

Wenn du diese Endungen erkennst, kannst du oft den richtigen Artikel erraten.

---

## Feminine Endungen

Wörter mit folgenden Endungen sind häufig **feminin (die)**.

### Häufige Endungen

- **-ion**
- **-ity**
- **-ik**
- **-ie**
- **-age**

### Beispiele

| Wort          | Geschlecht |
| ------------- | ---------- |
| die Nation    | feminin    |
| die Community | feminin    |
| die Musik     | feminin    |
| die Strategie | feminin    |
| die Garage    | feminin    |

### Tipp

Wenn ein Wort **wie eine typische feminine deutsche Endung klingt**, ist der Artikel oft **die**.

---

## Maskuline Endungen

Viele englische Lehnwörter, die **Personen oder Rollen beschreiben**, sind **maskulin (der)**.

### Häufige Endungen

- **-er**
- **-or**

### Beispiele

| Wort         | Geschlecht |
| ------------ | ---------- |
| der Manager  | maskulin   |
| der Computer | maskulin   |
| der Scanner  | maskulin   |
| der Blogger  | maskulin   |

### Tipp

Wenn das Wort **eine Person, einen Beruf oder eine Maschine beschreibt**, die etwas tut, ist der Artikel häufig **der**.

Beispiele:

- der Trainer  
- der Player  

---

## Neutrale Endungen

Viele Lehnwörter werden **neutral (das)**, besonders wenn sie **Dinge, Aktivitäten oder abstrakte Konzepte** beschreiben.

### Häufige Endungen

- **-ment**
- **-um**
- **-ing**

### Beispiele

| Wort          | Geschlecht |
| ------------- | ---------- |
| das Apartment | neutral    |
| das Museum    | neutral    |
| das Marketing | neutral    |
| das Training  | neutral    |

### Tipp

Englische Wörter mit **-ing** sind im Deutschen **sehr häufig neutral**.

Beispiele:

- das Meeting  
- das Timing  
- das Shopping  

---

# 3. Technologie- und Digitalwörter sind oft „das“

Viele moderne technische Begriffe oder digitale Lehnwörter verwenden **das**.

### Beispiele

| Wort           | Geschlecht |
| -------------- | ---------- |
| das Internet   | neutral    |
| das Smartphone | neutral    |
| das Tablet     | neutral    |
| das Display    | neutral    |

### Tipp

Wenn das Wort **ein modernes Gerät, Software oder ein digitales Konzept beschreibt**, ist **das** oft eine gute Vermutung.

---

# 4. Das Geschlecht kann auch von der Kategorie abhängen

Manchmal hängt der Artikel von der **Bedeutungskategorie des Wortes** ab.

---

## Essen und Getränke

Viele englische Lebensmittelbegriffe sind **neutral**.

Beispiele:

- das Sandwich  
- das Popcorn  
- das Steak  

Es gibt jedoch Ausnahmen:

- der Burger  
- der Whisky  

---

## Sportarten

Viele Sportarten sind im Deutschen **neutral**.

Beispiele:

- das Tennis  
- das Basketball  
- das Yoga  

---

# 5. Die Pluralform als Hinweis

Englische Lehnwörter bilden im Deutschen häufig den **Plural mit -s**.

Beispiele:

- die Jobs  
- die Teams  
- die Clubs  

Das **bestimmt jedoch nicht das Geschlecht im Singular**.

Beispiele:

| Singular | Plural    |
| -------- | --------- |
| der Job  | die Jobs  |
| das Team | die Teams |
| der Club | die Clubs |

---

# 6. Häufige Ausnahmen

Einige Wörter folgen **keinen klaren Regeln** und müssen einfach gelernt werden.

### Beispiele

| Wort       | Geschlecht |
| ---------- | ---------- |
| der Job    | maskulin   |
| der Laptop | maskulin   |
| das Event  | neutral    |
| das Team   | neutral    |
| die Show   | feminin    |

### Tipp

Lerne **den Artikel immer zusammen mit dem Substantiv**.

Beispiele:

- der Job  
- das Team  
- die Show  

---

# 7. Die beste Lernstrategie

Anstatt Wörter allein zu lernen, solltest du sie **immer zusammen mit ihrem Artikel** lernen.

### Beispiel

Weniger effektiv:

Computer

Besser:

der Computer

Noch besser:

Der Computer ist schnell.

Wenn du das Wort **in einem ganzen Satz** lernst, merkt sich dein Gehirn den Artikel leichter.

---

# Kurze Zusammenfassung

| Muster                             | Typisches Geschlecht |
| ---------------------------------- | -------------------- |
| Gleich wie das deutsche Äquivalent | unterschiedlich      |
| Endungen -ion, -ie, -ik            | die                  |
| Endungen -er, -or                  | der                  |
| Endungen -ment, -ing               | das                  |
| Technologie und Geräte             | oft das              |

---

# Letzter Tipp für Lernende

Wenn du **ein neues englisches Lehnwort im Deutschen** siehst, gehe am besten so vor:

1. Denke an das **deutsche Äquivalent**.
2. Achte auf die **Wortendung**.
3. Überlege, zu welcher **Kategorie** das Wort gehört (Technologie, Sport, Essen usw.).
4. Wenn du unsicher bist, ist **das oft eine gute Vermutung**.

Selbst Muttersprachler sind sich bei neuen Lehnwörtern manchmal unsicher – Fehler zu machen gehört also ganz normal zum Lernprozess.
